# zerobuilder-api

A library of the core functionality of zerobuilder.
It is intended for use in other annotation processors.

### Usage

See tests.

### Maven

````xml
<dependency>
    <groupId>com.github.h908714124</groupId>
    <artifactId>zerobuilder-api</artifactId>
    <version>1.483</version>
</dependency>
````
